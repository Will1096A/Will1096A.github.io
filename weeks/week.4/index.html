Forms

Form attributes: form action sends the completed form to a target website, the method command has two forms, get and post; get is the public method to get information from your form, while post is the private method to access your forms, labels are the start of the form, all labels need an input id (input is what the viewer fills in the form) which can have different types: passwords, dates, emails, colors, and text (which is the default), /placeholder= adds placeholder text within the form input, /textarea commands add a text area input used for larger inputs such as questions, the select command provides a dropdown menu of selectable options as an input with /option commands as the choices, /selected command adds the option value that its in as the default option, checkbox types adds a checkbox input, radio types add circle button inputs which let the viewer select only one choice, button commands add a clickable button which sends the form to the target website
