Web Design Notes

Various notes about coding in my web design foundations course:
/a href=
Commands direct the viewer to a part on the site or another page
/sub
Adds subscript below the line: this text is subscript
/sup
Adds superscript above the line; this is super script
/blockquote
Adds standalone quotes with their own citations outside of a paragraph
/hr
A thematic break in text as seen above, used for scene or topic changes
